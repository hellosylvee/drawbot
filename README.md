# Drawbot

A Slackbot that offers random things for you to draw.

## Prerequisites

```bash
$ brew install nodejs
```

## Setup

```bash
$ git clone git@github.com:spilth/drawbot.git
$ cd drawbot
$ npm install
$ BOT_API_KEY=your_api_key node bin/bot.js
```
